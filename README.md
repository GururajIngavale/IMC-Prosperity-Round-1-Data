# 📌 About IMC Prosperity 4

IMC Prosperity 4 is a global algorithmic trading competition organized by IMC Trading. It is designed for students to experience real-world trading through a simulated market environment.

Participants compete in 5 rounds, each consisting of:

Algorithmic trading (Python-based strategies)
Manual trading challenges

The objective is to maximize profit in a virtual market, applying concepts like market making, pricing, and risk management. The competition mirrors real trading scenarios and serves as both a learning platform and a recruitment opportunity.

## 📊 About This Dataset (Round 1 Data)

This repository contains Round 1 data from the IMC Prosperity competition, which is used to:

Simulate trading environments
Develop and test algorithmic strategies
Perform exploratory data analysis (EDA)

## 🔍 Dataset Characteristics

- Time-series trading data (prices, bids, asks, trades)
- Multiple synthetic financial instruments
- Designed to reflect real market dynamics such as:
- Market making opportunities
- Mean-reverting assets
- Noisy / volatile instruments
